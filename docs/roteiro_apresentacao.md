# Roteiro manual de testes

1. Executar o projeto:

```bash
python main.py
```

2. Informar um nome para o jogador.
3. Verificar se o sistema lista os 4 tópicos.
4. Escolher um tópico pelo número.
5. Responder uma pergunta corretamente.
6. Conferir se o sistema mostra que acertou e exibe a alternativa correta.
7. Responder uma pergunta incorretamente.
8. Conferir se o sistema mostra que errou, exibe a alternativa correta e mostra a explicação da alternativa escolhida.
9. Testar uma entrada inválida, como `E`, e verificar se o sistema pede uma nova resposta.
10. Responder as perguntas restantes.
11. Verificar o resultado final com acertos, erros, percentual e feedback geral.
12. Escolher jogar novamente com `S`.
13. Iniciar uma nova partida e depois escolher sair com `N`.
